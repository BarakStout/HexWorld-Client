# HexWorld-Client
Client code to connect to HewWorld server

## Installation

1. Click on _Clone or download_
2. Click on _Download Zip_
3. Wait for download to complete
4. *Unzip the files*
5. Open _index.html_ with a web browser
6. Type in the server's ip address
7. Have fun!
